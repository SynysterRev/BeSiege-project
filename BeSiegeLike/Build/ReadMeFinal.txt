Input de base :
Déplacement véhicule : ZQSD
Utilisation des ballons : W
Utilisation turbo : shift
Utilisation canon : espace
Possibilité de changer les inputs en appuyant sur le bouton Input ingame;

Pour sauvegarder le véhicule rentrer un nom dans le champ de texte et appuyer sur Save.

Echap pour mettre sur pause
Echap pour quitter le jeu une fois dans les menus

Pour mettre à jour le jeu, prendre les bundles qui sont dans le dossier "mises à jour" et les copier coller dans le dossier "StreamingAssets".
Pour importer de nouveaux véhicules, il faut mettre les sauvegardes dans le dossier "Vehicles" se trouvant dans "StreamingAssets".

Il se peut que lors de l'édition les objets se mettent à tourner lentement sur eux même, pour palier à cela il faut recliquer sur l'objet que l'on veut
dans l'UI de sélection.

Les blocs ont tous les mêmes propriétés, seule la texture est différente.

Les réacteurs et ballons n'ont pas de masse et ne déséquilibreront pas le véhicule mais le canon si, de même que les roues.

Possibilité de changer les forces appliqués sur les roues, les réacteurs et les ballons grâce à la petite réglette multiplie la valeur par la valeur
de base de la force. (si non touché utilisation des valeurs de bases)

A la fin de chaque niveau il faut retourner au menu pour en sélectionner un autre.
